SleekBot by Kev

Bot framework using the SleekXMPP library

Sleekbot is written in Python, using the SleekXMPP library. All features are implemented as plugins (in a similar fashion to SleekXMPP). Currently interesting plugins which are present, or under development include: * A 'ping' bot. * RSS feed announcements. * A '!seen' command for tracking users in a muc. * A XEP plugin which allows users to query the XMPP Extension Protocols. * An interface allowing the bot to run Eggdrop TCL scripts (allowing access to a large number of existing IRC scripts). Please drop by the XMPP conference sleek@conference.psi-im.org for a chat (timezones allowing).

Dependencies

http://github.com/fritzy/SleekXMPP

Install

Install SleekXMPP. SleekBot will run in-situ.
